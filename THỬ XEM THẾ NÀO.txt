HIỂN THỊ BẢNG ĐIỂM CHO PHỤ HUYNH

Yêu cầu về db: store procedure trả về bảng gồm những giá trị sau: 
+ id học sinh
+ điểm
+ hệ số(weight)
+ tên điểm
+ id loại điểm (grade_component_id)
+ tên môn
+ id môn (subject id)

về front end:
có 3 biến mảng (array):
+ mảng chứa danh sách mà sp trả về
+ mảng chứa các môn
+ mảng chứa các loại điểm (tên điểm)

tạo bảng front end với hàng đầu tiên là cá loại điểm, cột đầu tiên (các cell đầu tiên của từng hàng) là tên môn

mỗi cell chứa điểm sẽ có id là kết hợp của tên môn và tên loại điểm -> khi cần lấy giá trị trong mảng chứa danh sách tất cả điểm thì chỉ cần tìm object nào mà id của cell chứa cả 2 là tên môn và tên loại điểm
Ví dụ: id cell là MATH_15mins -> tìm object trong mảng tương ứng

này là chưa tính đến điểm kiểu hk1 hk2 thì cái đấy từ từ, cần thì tạo hẳn 2 bảng là bảng hk1 và hk2, hoặc thêm thành phần là hk vào trong bảng bên db.
